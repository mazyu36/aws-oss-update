## 概要

このリリースはメンテナンスリリースです。新機能やバグ修正は含まれておらず、以前のリリースで追加された bundling 関連の変更が revert されています。

**リリース:** [v2.258.1](https://github.com/aws/aws-cdk/releases/tag/v2.258.1)

## 変更内容

### Revert: bundling における docker image キャッシュチェックの削除 ([#38116](https://github.com/aws/aws-cdk/pull/38116))

- 以前のリリースで追加された [#37951](https://github.com/aws/aws-cdk/pull/37951)「chore(bundling): check if docker image is cached before building」の変更が revert されました。
- この変更は bundling 処理において、docker image のビルド前にキャッシュ済みかどうかをチェックする内部的な改善でしたが、何らかの問題があったため取り消されています。
- ユーザーの CDK アプリケーションのコードに影響する変更ではありません。

## まとめ

v2.258.1 は内部的な変更の revert を行ったメンテナンスリリースです。新機能やバグ修正、API の変更は含まれていないため、特に注意すべき移行作業はありません。
